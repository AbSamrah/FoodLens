# Welcome to your Expo app 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

### Running on your phone

1. Ensure your computer and phone are on the same Wi‑Fi network (or use the tunnel option).

2. Install dependencies and start with the tunnel option (recommended if devices are on different networks):

```bash
npm install
npm run start:phone
```

3. Open the Expo Go app on your phone and scan the QR code shown in the terminal or Expo DevTools. For iOS devices you may need to use the camera or the built-in QR scanner in Expo Go.

Notes:

- Use `npm run start:lan` to start with LAN (faster) if your phone can reach your machine on the same network.
- If you need native modules not supported by Expo Go (custom native code), create a development build or use a device build: https://docs.expo.dev/develop/development-builds/

In the output, you'll find options to open the app in a

- [development build](https://docs.expo.dev/develop/development-builds/introduction/)
- [Android emulator](https://docs.expo.dev/workflow/android-studio-emulator/)
- [iOS simulator](https://docs.expo.dev/workflow/ios-simulator/)
- [Expo Go](https://expo.dev/go), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://docs.expo.dev/router/introduction).

> **Tip:** In the camera screen you can either take a new photo or choose an existing one from your library. Permissions for camera and photo library are requested automatically when the screen loads.

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.
